# **App Name**: Kompost Mixer

## Core Features:

- Google Authentication: User authentication using Google Sign-In, enabling secure access to the application.
- Track List View: Track List View: Displays music tracks in a searchable and filterable list.
- Audio File Upload: File Upload: Enables users to upload audio files (mp3, wav, m4a, flac) to Google Cloud Storage.
- Real-time Form Validation: Form Validation: Provides real-time feedback on form inputs to ensure data accuracy.
- Error messages: Displays error messages for different scenarios.
- Metadata Suggestion: Automatic audio analyzer tool that helps suggest BPM and Key metadata for uploaded tracks, improving categorization accuracy.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to establish a professional yet creative ambiance.
- Background color: Light grayish-blue (#E8EAF6), offering a subtle contrast that ensures readability.
- Accent color: Soft purple (#9575CD), strategically applied on interactive elements.
- Body font: 'PT Sans', sans-serif, for body text as well as headlines.
- Utilize a set of streamlined, outline-style icons that are easy to interpret, matching the rest of the application’s design.
- Adopt a grid-based layout for all sections.
- Incorporate subtle transition effects that won’t detract users from the main purpose of the website.
